# GAOP
🌍 GAOP (Gusto Atlas Open Source Project)
O GAOP é a base open source do Gusto Atlas.
Ele contém toda a infraestrutura responsável pelo funcionamento do motor de busca.
Recursos:
🌐 Gerenciamento de até 200 instâncias SearXNG.
⚡ Seleção automática das melhores instâncias.
🟢 Detecção de instâncias online.
🔴 Detecção de instâncias offline.
🔄 Substituição automática de instâncias indisponíveis.
🏆 Ranking inteligente.
❤️ Monitoramento contínuo das instâncias.
📥 Coleta de até 20 resultados por instância.
🧹 Remoção de resultados duplicados.
⭐ Filtro de qualidade.
📊 Organização dos resultados por relevância.
📁 Arquitetura modular.
O GAOP não inclui:
🎨 Interface gráfica.
🔐 Sistema de login.
👤 Contas de usuário.
